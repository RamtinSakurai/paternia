design notes

symbol meaning

5 つの金の点はビッグファイブ(O · C · E · A · N)の 5 軸。これを結ぶ薄い五角形は、あなたの性格が描く「模様」のシルエット。中心の1 つのドットは、観測の中心 = 今のあなた自身。

外側の円は天体観測の枠。全体で「性格の星図(atlas)」を象徴する。

color rule

  • Primary: Gold (#D8BB89) + Gold Bright (#EAD1A0) 中心
  • Monochrome: Ivory(ダーク背景) / Ink Walnut(#2F2418)(ライト背景)
  • 外側 stroke にグラデーション(#EAD1A0 → #A88B55)を使えるのは高解像度時のみ(48px 未満はフラット色)

spacing

ロゴ周囲には最低でも、マーク幅の 25% の余白を確保する(clear space)。

minimum sizes

  • Icon: 16×16 px まで(16px 以下は非推奨、star radial details が潰れる)
  • Wordmark: 高さ 14px まで
  • Lockup 縦: 56px 以上推奨

file output plan

  • public/favicon.svg — 64px base(既存 favicon.ico を置換)
  • public/og/paternia-logo-gold.svg — OGP / sharing
  • src/components/PaterniaLogo.tsx — React component(variant prop で切替)
  • TikTok 用: 1080×1080 サイズで PNG エクスポート(プロフィール画像)
  • note 用: ヘッダー画像(1280×670)+ アイコン(200×200)
